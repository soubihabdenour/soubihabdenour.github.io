## ğŸŸª Part 5: Do-While Loop & Input-Based Loops

### ğŸ§  Concept

- `do-while`Â ensures at least one execution
- Useful for input validation or repeated tasks

### ğŸ§ª Example: Sum Until Negative

```c
#include <stdio.h>

int main(void) {
    int num;
    int sum = 0, count = 0;

    do {
        printf("Enter a non-negative number: ");
        scanf("%d", &num);

        sum += (num >= 0) * num;
        count += (num >= 0);
    } while (num >= 0);

    printf("You entered %d non-negative numbers.\n", count);
    printf("Their sum is %d.\n", sum);

    return 0;
}
```
